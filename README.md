# dsa
**Data Structures and Algorithms**

A simple tutorial on the common data structures and algorithms used in coding interviews, with full implementation in Python3.

Includes the following:

## Data Structures
### Linear
- Native
  - Lists
  - Tuples
  - Sets
  - Dicts   
- Numpy Arrays
- Matrices and Tensors (TensorFlow)
- Stacks, Queues, and Deques
- Linked Lists
- Doubly Linked Lists
- Hash Maps
### Nonlinear
- Binary Trees
- Binary Search Trees
- Heaps
- Graphs and Networks 
- Tries

## Algorithms
### Sorting
- Quick Sort
- Merge Sort
- Heap Sort
- Insertion Sort
- Radix Sort
### Searching
- Linear Search
- Binary Search
### Traversal
- Depth-First Search
- Breadth-First Search
- Kruskal’s Algorithm
- Prim’s Algorithm
- Dijkstra’s Algorithm
### Dynamic Programming
- Fibonacci Sequence
